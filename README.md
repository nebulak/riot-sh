# Features
## Apps & App Hardening

  - TorBrowser
  - OpenPGP and tor for email(thunderbird + enigmail + torbirdy)
  - Onion-Gajim with OMEMO support(Gajim over tor using torjail)
  - KeepassXC
  - restic for encrypted backups
  - syncthing
  - gocryptfs
  - Magic Wormhole for encrypted file transfer
